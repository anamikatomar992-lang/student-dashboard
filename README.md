# student-dashboard
📚 Study Sanctuary: A Glassmorphic Student Ecosystem
Project Overview
Study Sanctuary is a modern, high-fidelity student dashboard designed to balance aesthetic appeal with academic productivity. Moving away from cluttered, high-contrast interfaces, this project utilizes Glassmorphism and a Library-inspired color palette to create a "Study Sanctuary"—a calm, focused digital environment for learners.

Technical Implementation
This project was built to demonstrate proficiency in modern frontend architecture and data visualization:

UI/UX Engineering: Implemented a complex glassmorphic design using CSS backdrop-filter and custom properties (CSS Variables) to manage a sophisticated "Soft Indigo & Warm Ivory" theme.

Data Visualization: Integrated Chart.js to render dynamic study trends, utilizing linear gradients and cubic bezier curves for a premium look.

Vector Graphics: Utilized Lucide-React for a consistent, scalable, and lightweight iconography system.

Responsive Architecture: Engineered using a mobile-first approach with CSS Grid and Flexbox to ensure seamless layout transitions across devices.

Key Features
Dynamic Task Manager: A streamlined to-do system with interactive state management.

Academic Progress Analytics: Real-time visualization of study hours and attendance metrics via custom SVG circular progress bars.

Cozy Library Aesthetic: Depth-of-field background effects and warm ambient lighting simulations to reduce digital eye strain.

Multi-Page Navigation: Separate logic for an entry portal (index.html) and the main workspace (dashboard.html).
